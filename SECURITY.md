# Security Policy

## Security Notice

We take the security of this project seriously. If you discover a potential
vulnerability, we encourage you to let us know.

## Reporting Vulnerabilities

Please report security issues confidentially. You may contact us via email or
through the project page. We aim to respond to reports in a timely manner, but
cannot guarantee specific response times.

## Supported Versions

We strive to support the current and previous major releases. Older versions may
not receive security updates.

## Additional Information

Further details and guidelines may be added over time. There is no guarantee of
a particular response time or the release of security fixes.
